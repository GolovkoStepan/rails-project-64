# Коллективный блог

Аналог habr.com. Каждый пользователь может создать запись и опубликовать ее в одной из общих категорий.
Другие пользователи ставят оценки и комментируют добавленные посты.

### Установка зависимостей
```bash
make setup
```

### Запуск приложения
```bash
make start
```

### Запуск тестов и проверок кода
```bash
make check
```

Развернутое приложение: https://rails-project-64-ij6m.onrender.com
